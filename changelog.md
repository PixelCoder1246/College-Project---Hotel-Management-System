# Changelog

All notable changes to this project will be documented in this file.

## [0.0.0] - 2026-04-07
### Added
- Initial monorepo directory structure:
  - `backend/` for server-side code.
  - `frontend/` for client-side code.
  - `services/` for standalone business logic or microservices.
  - `utils/` for shared utilities and helper functions.
- `.gitkeep` files to preserve empty directories.
- `changelog.md` for tracking project changes.
- Project structure documentation in `readme.md`.
